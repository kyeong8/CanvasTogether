**멀티 접속하여 서로 채팅과 캔버스 그림을 공유하는 프로그램**

***

### 사용 방법
1. Client 폴더에 Database 폴더를 만든 후 cvtDB.db 파일을 생성한다.
2. Server와 Client를 키고 회원가입 후 로그인을 한다.
3. 로비에서 방 생성 후 접속하여 재밌게 그림을 그린다.
